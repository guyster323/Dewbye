HOW TO RUN THIS WEB APP
-----------------------

Option 1: Using Python (Recommended)
------------------------------------
1. Install Python from https://www.python.org/downloads/
2. Double-click 'start_server.bat' (Windows)
   OR run 'start_server.ps1' in PowerShell
3. Open browser and go to: http://localhost:8000

Option 2: Using VS Code Live Server
-----------------------------------
1. Install VS Code
2. Install "Live Server" extension
3. Open this folder in VS Code
4. Right-click on index.html -> "Open with Live Server"

Option 3: Using Node.js http-server
-----------------------------------
1. Install Node.js from https://nodejs.org/
2. Run: npm install -g http-server
3. In this folder, run: http-server
4. Open browser and go to: http://localhost:8080

Option 4: Deploy to Web Hosting
-------------------------------
Upload all files in this folder to any web hosting service:
- GitHub Pages (free)
- Netlify (free)
- Vercel (free)
- Firebase Hosting
- AWS S3
- Any web server

IMPORTANT NOTES
---------------
- This is a Single Page Application (SPA)
- CORS may block API requests if running from file:// protocol
- Always use a web server (http:// or https://)
- Do NOT open index.html directly in browser

FEATURES
--------
- Weather-based condensation risk analysis
- Historical data analysis
- Interactive charts with zoom/pan
- PDF report generation (English)
- CSV data export
- Multi-building type support
