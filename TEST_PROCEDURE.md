## 9. 알려진 이슈

### 9.1 현재 알려진 제한사항
1. **기상청 API**: API 키가 필요하며, 키가 없으면 Open-Meteo만 사용
2. **알림**: 인앱 알림만 지원 (푸시 알림 미구현)
3. **오프라인 모드**: 캐시된 데이터만 표시 가능

### 9.2 향후 개선 예정
- [ ] 푸시 알림 구현 (firebase_messaging)
- [ ] 오프라인 모드 강화
- [ ] 위젯 지원 (홈 화면 위젯)
- [ ] 다국어 지원
